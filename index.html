Single Page Application Basics

  • Single index.html page
  • Dynamically updates view/content based on user interaction
  • Better User Experience

Single Page Application Cons

  • Anyone browsing without JavaScript enabled will not get all the functionality
  • Search Engine Optimization (SEO)
  • It's not very common for people to browse without JS enabled anymore but it can happen
  • Search engines have a hard time crawling SPAs
    • Google has gotten better
    • Sometimes have to point the bots to a rendered page (something like rendered.js
